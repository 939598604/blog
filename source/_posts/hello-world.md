## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Run server

``` bash
$ hexo server
```

``` bash
$ hexo generate
```

``` bash
$ hexo deploy
```

More info: [Deployment](https://hexo.io/docs/deployment.html)
